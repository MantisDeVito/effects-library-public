# Callum's Effects Library — Public Preview

Browse-only mirror of Callum's video-effects reference deck.

## Pages

- `search.html` — searchable catalogue (start here)
- `recipes.html` — pre-paired overlay + SFX combos
- `transitions.html` — transition cards
- `entrances.html` — appear / disappear motion vocab
- `lower-thirds.html` — name-and-title cards
- `colours.html` — colour-grade recipes
- `logos.html` — logo reveal patterns
- `mograph.html` — motion-graphic patterns
- `glitch.html` — glitch / corruption looks
- `captions.html` — caption layouts
- `atmosphere.html` — fog, rain, smoke, lens dust, etc.

## Public-preview limitations

- Card previews that hit the offline shared-library (greenscreens, music, SFX, big mograph clips) will not load.
- The **Drop into Premiere** buttons on `recipes.html` are greyed out — that feature requires a local helper server.
- A few topnav menu items point at local-drive shortcuts and won't resolve.

Everything coded inside the page (CSS effects, in-browser canvas processing, included sample MP4 clips) works.
